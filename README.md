# Empower Foundation

The official website for **Empower Foundation** — a student-led 501(c)(3) nonprofit based in Orange County, CA, focused on education and sustainable community development.

It's a static site — plain HTML, CSS, and JavaScript with no build step or framework.

## Pages
- `index.html` — home
- `about.html` — about us
- `programs.html` — Education & Sustainable Development
- `impact.html` — impact & stats
- `partnership.html` — partners (incl. Boys & Girls Club of Stanton)
- `gallery.html` — photo gallery
- `volunteer.html` — volunteering & membership
- `contact.html` — contact form

## Run locally
Any static file server works. For example, with Python:

```bash
python -m http.server 4321
```

Then open <http://localhost:4321>.

## Assets
- `images/` — photos
- `logo.png` / `logo-mark.png` — brand mark
